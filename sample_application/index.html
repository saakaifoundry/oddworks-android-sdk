Houston, We Have Lift Off!

For your convenience, we set up a simple example application that connects to a ficticious NASA account at Oddworks, fetches data using a few of the SDK Data Models outlined further on in our Developer Guide, and displays that data to users in a visually pleasing application.

Getting up and running with Odd Networks on Android is as easy as following these 5 simple steps.

Step 1

Clone the GitHub project to your local machine.

Odd Sample Apps

    $ git clone git@github.com:oddnetworks/odd-sample-apps

Step 2

Open the application in Android Studio.

Step 3

Once Gradle has synced the dependencies, you are ready to start the application.

Step 4

Select your deployment target.

Step 5

Enjoy some NASA space goodness!

What's Next?

You're up and running, but that's just the beginning. To explore more of what Oddworks has to offer and get your own account, head over to oddnetworks.com and get started!

For More Information

Head over to the Getting Started section of this Developer Guide which will give an overview of Oddworks and take you through the Oddworks SDK for Android. Also check out the Oddworks SDK for Android Java documentation if you really like to dig deep!
